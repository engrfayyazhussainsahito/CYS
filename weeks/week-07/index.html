Week 7

Network Security

Week 7 teaches networks from first principles and then connects those foundations to firewalls, segmentation, monitoring, routing risk, secure remote access, IDS/IPS, and safe defensive analysis.

Estimated reading time12–20 hours
Difficulty levelBeginner → MS foundation
Required backgroundWeeks 0–6 recommended
Major outputNetwork diagram and defensive flow review

Overview

Networks are the paths through which systems communicate. This week starts with hosts, packets, IP addresses, ports, DNS, TCP/UDP, routing, and logs, then moves toward firewall policy, segmentation, traffic analysis, IDS/IPS, VPNs, wireless basics, and secure network design.

Learning Objectives

  • Explain how packets move through networks.
  • Read basic local network configuration safely.
  • Describe firewalls, routing, segmentation, VPNs, IDS/IPS, and network telemetry.
  • Design simple secure network zones and allowed flows.
  • Use authorized local labs to inspect traffic and write a defensive report.

Required Background

No advanced networking background is required. Students should understand basic Linux commands, Python, web concepts, identity, and security goals from earlier weeks. Each lesson reintroduces prerequisite concepts before using them.

Lessons

Lesson 7.1: Network Security from First Principles

Hosts, packets, addresses, ports, TCP, UDP, DNS, routing, NAT, firewalls, segmentation, monitoring, availability, and a safe local network lab.

Lesson 7.2: TCP/IP, DNS, and Routing Security

Packet delivery, TCP and UDP, DNS resolution, DNSSEC, routing tables, BGP risk, route leaks, route hijacks, and defensive telemetry.

Lesson 7.3: Firewalls, Segmentation, and Zero Trust

Firewall policy, stateful inspection, host/network/cloud firewalls, DMZs, segmentation, microsegmentation, and Zero Trust architecture.

Lesson 7.4: Traffic Analysis, IDS/IPS, and Detection

Packets, flows, DNS logs, IDS, IPS, signatures, anomaly detection, alert triage, false positives, tuning, and network security monitoring.

Lesson 7.5: VPNs, Wireless, and Secure Network Design

VPN tunnels, remote-access and site-to-site designs, split/full tunneling, WPA2/WPA3, 802.1X, RADIUS, guest isolation, IoT segmentation, and resilient secure network design.
