# Lesson 7.3: Firewalls, Segmentation, and Zero Trust

## Overview
Firewalls, segmentation, and Zero Trust from first principles.

## Major concepts
- Firewall rule anatomy
- Stateful and stateless filtering
- Deny by default
- Network zones and DMZs
- North-south and east-west traffic
- VLANs, ACLs, cloud security groups, WAFs
- Microsegmentation
- Zero Trust architecture
- Policy engine, policy administrator, policy enforcement point
- Rule review and residual risk

## Lab
Design and evaluate a fictional segmented university network using safe toy policies only.
