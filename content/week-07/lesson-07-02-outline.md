# Lesson 7.2: TCP/IP, DNS, and Routing Security

## Overview
TCP/IP, DNS, and routing security from first principles.

## Major concepts
- TCP/IP encapsulation
- IP addressing and headers
- TCP vs UDP
- DNS resolution, records, caching, TTL, DNSSEC, DNS privacy
- Routing tables, default routes, longest-prefix match
- BGP, route leaks, route hijacking, RPKI
- Defensive DNS and routing telemetry

## Lab
Safe local TCP/IP, DNS, and routing observation using localhost, local OS commands, and known documentation/test domains only.
