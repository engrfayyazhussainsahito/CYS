# Lesson 3.2 — Randomness, Keys, Hashes, and MACs

Core scope: entropy, CSPRNGs, keys, key lifecycle, key separation, hashes, password hashing, salts, peppers, MACs, HMAC, constant-time comparison, and secret leakage.

Lab: safe local token generation, SHA-256 file hashes, HMAC generation/verification, and password-hashing reasoning.

Primary references: NIST SP 800-90A Rev. 1, NIST FIPS 180-4, NIST FIPS 198-1, RFC 2104, NIST SP 800-57 Part 1 Rev. 5, Python secrets/hmac docs, OWASP Password Storage Cheat Sheet.
