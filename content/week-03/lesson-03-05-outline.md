# Lesson 3.5 — Crypto Misuse, Key Management, and Crypto Agility

## Core scope
- Cryptographic misuse
- Key management lifecycle
- KMS, HSM, secrets managers
- DEK, KEK, envelope encryption
- Key rotation and cryptoperiods
- Cryptographic inventory and CBOM
- Crypto agility and post-quantum migration readiness

## Safe lab
Students create a local fictional app folder, identify training-only insecure crypto patterns, build a crypto inventory, and write a remediation memo.
