# Lesson 2.5 — Privilege, Services, Logs, and System Hardening

## Purpose
Teach privilege, services/daemons, service accounts, listening ports, logs, audit logs, baselines, configuration drift, and hardening from first principles.

## Safe lab
Students collect read-only local service/log evidence, compare it against a fictional classroom-server baseline, analyze fictional logs with Python, and write a hardening recommendation report.

## Key outputs
- Local service inventory
- Listening-port review
- Fictional log-analysis output
- Privilege and service-account risk notes
- Hardening recommendation report
