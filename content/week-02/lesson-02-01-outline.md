# Lesson 2.1 — Operating Systems from First Principles

## Purpose
Teach operating systems from zero knowledge for cybersecurity students.

## Core path
Hardware → firmware → bootloader → kernel → system calls → processes → memory → filesystem → users/groups → permissions → services → logs → hardening.

## Safe lab
Local OS inventory, file permission inspection, process/service listing, and log review on the student's own lab machine only.
