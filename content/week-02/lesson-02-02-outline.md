# Lesson 2.2 — Users, Groups, and Identity

This lesson teaches digital identity from first principles: subjects, accounts, users, groups, service accounts, machine accounts, workload identities, identifiers, attributes, UIDs, GIDs, Windows SIDs, local accounts, directory accounts, privileged accounts, identity lifecycle, stale accounts, orphaned accounts, privilege creep, detection, and safe account-review labs.

Safe lab: observe only the learner's own local identity, use fictional account records, and produce an identity review report without modifying real accounts.
