# Lesson 2.4 — Authorization, Permissions, and Access Control Models

## Purpose
Teach authorization from first principles: subjects, objects, actions, context, policies, permissions, ACLs, Linux/Windows permissions, RBAC, ABAC, DAC, MAC, least privilege, object-level authorization, and access reviews.

## Safe lab
Students build a fictional access-control matrix, practice local file permissions, run a local Python policy evaluator, and write a least-privilege report. No real targets or production systems are used.

## Key outputs
- Permission interpretation notes
- Fictional access-control matrix review
- Python deny-by-default authorization evaluator
- Least-privilege recommendation report
