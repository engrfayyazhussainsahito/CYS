# Lesson 9.3 — Endpoint Hardening and Patch Management

## Purpose
Teach endpoint hardening and patch management from first principles with a defensive, evidence-based approach.

## Core topics
- Hardening vs patching
- Attack surface reduction
- Secure baselines
- Configuration drift
- Patch lifecycle
- Update rings
- Risk-based patch prioritization
- Patch compliance metrics
- Exceptions and compensating controls
- End-of-life software
- Rollback and recovery planning
- Safe endpoint review lab

## Lab
Safe Endpoint Hardening and Patch Review using only owned or authorized systems.
