# Lesson 10.5: Cloud Logging, Monitoring, and Incident Response

## Summary
Cloud logging, monitoring, and incident response from first principles.

## Major topics
- Telemetry, logs, metrics, traces, alerts
- Audit logs, control-plane logs, data-plane logs
- Identity, storage, network, workload, Kubernetes, application, and pipeline logs
- Centralized logging, retention, integrity, and access control
- Detection engineering, triage, timelines, containment, eradication, recovery
- Cloud incident reporting, runbooks, and lessons learned

## Lab
Fictional local JSONL cloud audit log triage using Python. No real cloud account, scanning, or unauthorized testing.
