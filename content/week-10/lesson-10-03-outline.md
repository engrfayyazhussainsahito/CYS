# Lesson 10.3: Containers, Images, and Kubernetes Security

Status: ready

This lesson teaches containers, images, registries, Dockerfile risk, Kubernetes architecture, Pods, nodes, service accounts, RBAC, security contexts, NetworkPolicy, admission control, runtime detection, and local-only manifest review. It includes all required lesson sections, Mermaid diagrams, mind map, math foundations, code examples, case studies, self-assessment, and a safe practice lab.
