# Lesson 4.5 — Testing, Fuzzing, CI/CD, and Release Integrity

## Scope
- Testing from first principles
- Unit, integration, end-to-end, regression, and security tests
- Fuzzing concepts and safe local fuzzing
- CI/CD pipelines and pipeline trust boundaries
- Runner security and CI/CD secret risks
- Release gates and release evidence
- Artifact hashes, signatures, provenance, and attestations
- Safe local release-evidence lab

## Safety
All demonstrations use local toy code only. Do not fuzz, scan, or test real systems without explicit written authorization.
