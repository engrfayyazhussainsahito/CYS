# Lesson 0.4 — Python for Security Learners

## Purpose
Teach Python from zero knowledge for safe cybersecurity automation and analysis.

## Major topics
- Programs, scripts, interpreters
- Variables and data types
- Conditionals, loops, and functions
- Files, paths, logs, JSON, and regular expressions
- Error handling and logging
- Virtual environments
- Safe subprocess patterns
- Localhost-only networking
- Defensive automation
- Secure coding habits
- Safe log-analysis lab

## Safety boundary
All examples use local files, sample data, or localhost only. No unauthorized scanning, exploitation, credential guessing, malware development, or public-target automation.
