# Lesson 0.5 — Git, Documentation, and Evidence Handling

## Purpose
Teach Git, technical documentation, evidence logs, hashes, chain of custody, provenance, and reproducible cybersecurity work from first principles.

## Core deliverables
- Git repository setup
- README
- Evidence log
- SHA-256 hash calculation
- Safe documentation lab
- Branch, merge, tag workflow

## Safety boundary
Local training files only. No real credentials, private keys, personal data, employer data, or third-party evidence.
